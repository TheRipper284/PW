Prototipo Móvil

Es una aplicación móvil desarrollada con Flutter que consta con una conexión a una base de datos en la nube NoSQL con los datos del cuidador del canino, la conexión a sensores para el monitoreo de pulso cardiaco, temperatura. En el que se resguardarán los datos de monitoreo de manera constante, para llevar un control sobre el canino.
